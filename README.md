# txt.wwew.tech

Локальный (client-side) инструмент для конвертации файлов в единый LLM-ready `.txt` без серверной обработки.

## Что реализовано

- Трёхпанельный UI в стиле современного SaaS/IDE.
- Левая панель: бренд, новый чат, история из `localStorage`, сворачивание, тема, полезные ссылки.
- Центр: поле запроса, drag-and-drop, загрузка файлов/папок, карточки вложений, превью-модалка.
- Правая панель: анонимный режим и настраиваемые фильтры (игнор папок и расширений).
- Клиентский парсинг `txt/code/json/csv/html/xml/yaml`, `pdf`, `docx`, `zip`.
- Приблизительная оценка токенов и экспорт итогового контекста в `.txt`.

## Стек

- Next.js (App Router, TypeScript)
- Tailwind CSS v4
- `next-themes`, `lucide-react`
- `mammoth` (DOCX), `pdfjs-dist` (PDF), `jszip` (ZIP)

## Запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:3000`.

## Важно

- Все вычисления выполняются в браузере.
- В анонимном режиме история не записывается в `localStorage`.
